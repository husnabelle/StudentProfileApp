## Project Description
This project is a web-based system built using JSP, Servlets, HTML database.  
It allows students to create, view, edit, and delete their profiles.  
The system connects with a database to store all student information permanently.

## Implemented Features
- Add new student profile
- View all student profiles
- Edit existing profile
- Delete profile

## Database
The database script is provided in `database.sql`.  
It creates the `PROFILE` table with the following columns:
- STUDENTID
- NAME
- PROGRAMME
- EMAIL
- HOBBIES
- INTRODUCTION
